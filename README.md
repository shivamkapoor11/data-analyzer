# Data Analyzer Agent

AI-powered Data Analysis Tool!

## Live App
https://data-analyzer-pfzmxnyfzkr4tgzz3x22b5.streamlit.app/

## Features
- Koi bhi Excel ya CSV upload karo
- Auto charts generate hote hain (Bar, Line, Scatter, Histogram)
- AI key insights nikalti hai
- Trends detect karta hai
- Business recommendations deta hai

## Built With
- Python
- Groq AI (Llama 3.3)
- Streamlit
- Plotly
- Pandas

## How To Use
1. App kholo
2. Excel ya CSV upload karo
3. Charts aur AI analysis dekho!
